# uf_ulog_macro

Derive macros for `uf_ulog`.

This crate provides `#[derive(ULogData)]` and `#[derive(ULogRegistry)]` to generate message encoding and registry metadata.

- Crate: `uf_ulog_macro`
- Repository: <https://github.com/jettify/uf-ulog>
- Docs: <https://docs.rs/uf_ulog_macro>

Macro usage is optional; `uf_ulog` also supports manual trait implementations.
